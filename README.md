 Features

Finds subdomains

Detects live hosts

Collects URLs and endpoints

Finds JavaScript files and hidden APIs

Saves everything into one report file

Installation

Clone and setup:

git clone https://github.com/<your-username>/BashFist.git
cd BashFist
chmod +x bashfist.sh

Usage

Run with:

./bashfist.sh example.com
